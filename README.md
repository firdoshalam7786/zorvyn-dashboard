# 💰 Finance Dashboard UI

A clean and interactive finance dashboard built using **React + Tailwind CSS**.
This project helps users track their income, expenses, and overall financial activity in a simple and visual way.

---

## Deploy Project
[ Live Demo ](https://zorvyn-dashboard-green-ten.vercel.app)

Live Project Url: https://zorvyn-dashboard-green-ten.vercel.app

## Features

* **Dashboard Overview**

  * Total Balance, Income, and Expenses cards
  * Balance Trend (Line Chart)
  * Spending Breakdown (Pie Chart)

* **Transactions Section**

  * View all transactions (date, amount, category, type)
  * Search and filter functionality
  * Responsive table (card layout on mobile)

* **Role-Based UI**

  * Admin → Add, Edit, Delete transactions
  * Viewer → Read-only access

* **Insights Panel**

  * Top spending category
  * Monthly comparison
  * Helpful tips

* **Responsive Design**

  * Works on mobile, tablet, and desktop

---

## Tech Stack

* React.js
* Tailwind CSS
* Chart.js / React Chart.js 2
* Context API (State Management)
* react-icons(For Icon)

---

## Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/firdoshalam7786/zorvyn-dashboard
```

2. Go to project folder:

```bash
cd finance-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm run dev
```

## Approach

I focused on building a simple and clean UI that is easy to understand and use.

* Used **component-based structure** for better readability
* Managed state using **Context API**
* Added **debounced search** for performance
* Designed **mobile-first responsive layout**
* Focused on **clean UI and smooth user experience**

---

## Key Highlights

* Clean and modern dashboard design
* Fully responsive layout
* Smooth filtering and search
* Role-based UI simulation
* Optimized performance using memoization
